📋
计费规则快速参考

📦 入库计费

  • 三边和分4档：≤55 / ≤130 / ≤200 / >200cm
  • 收费合计 = 计费单价 × 上架量（已上数量）
  • 例：单价 0.16 × 上架量 50 = ¥8.00
  • 三边和=0 → 计费0并提醒补充尺寸
  • 西之月跨境 → 股东价；其余 → 公开价

🚚 出库计费

  • 计费重量 = max(实重kg, 体积重kg)
  • 体积重 = 长×宽×高 ÷ 6000；重量(g) ÷ 1000 = 实重(kg)
  • 同订单（关联单号）多SKU合并计费：按出库数量合计
  • ≤4件：只收1件首件费（取最重SKU的档位），其余免
  • ＞4件：1件首件 + 免3件，第5件起按各SKU各自重量档位 × 出库数量叠加续件费
  • 续件费档位：≤1kg→¥0.50；1~3kg→¥0.70；3~10kg→¥0.90
  • 超尺寸自动递归升档（兼顾重量+单边限制）
  • 通过关联单号/订单编号匹配子表（订单信息）补全SKU尺寸重量
  • 订单号无匹配时自动用SKU在子表搜索；SKU含"*"后缀自动去掉再匹配
  • 缺尺寸→仅实重计费；缺重量→仅体积重；二者均缺→最低档
  • 数量不一致时底部显示异常提醒表格
  • 西之月跨境 → 股东价；其余 → 公开价

↩️ 退货计费

  • 退件入库费：固定 2.00元/单，同订单/关联单号只收1次
  • 退件上架费 = 计费单价 × 上架量，档位与入库费同规则
  • 重复计费行 → 红色标注
  • 西之月跨境 → 股东价；其余 → 公开价
